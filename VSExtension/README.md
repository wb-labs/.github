# 💻 VS Code Extensions

Bringing the **WB-Labs** ecosystem directly into your development workflow.

## 🛠️ Featured Tool: [VSExtension](https://github.com/wb-labs/VSExtension)

The official VS Code extension for the WBC-UI2 ecosystem, designed to provide first-class DX (Developer Experience).

### Core Features (Planned)

- **Live Preview**: Real-time rendering of `.wbc`, `.json`, and `.md` files using the WBC engine.
- **IntelliSense**: Smart completion for WBC props, lifecycle hooks, and pipe syntax.
- **Diagnostics**: Validation of WBC configuration files and data schemas.
- **Scaffolding**: One-click generation of new WBC components and project structures.
