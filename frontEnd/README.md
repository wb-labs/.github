# 🎨 Front-End Products

This directory contains public documentation for the frontend libraries and frameworks developed by **WB-Labs**.

## 🚀 Featured Product: [WBC-UI2](./wbc-ui2/)

**WBC-UI2** is our flagship component orchestration engine for Vue.js. It transforms raw data into interactive, production-ready interfaces with zero boilerplate.

### Sub-packages
- **`wbc-ui2`**: The Core Engine.
- **`@wbc-ui2/code`**: Advanced syntax highlighting and live editors.
- **`@wbc-ui2/chart`**: Low-code Chart.js and ECharts wrapper.
- **`@wbc-ui2/latex`**: TeX/LaTeX mathematical formula rendering.
- **`@wbc-ui2/mermaid`**: Text-to-diagram rendering.
