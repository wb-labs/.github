# 🧩 Browser & VS Code Extensions

Enhancing developer productivity directly within the tools they use every day.

## 🌐 Chrome Extensions

Tools for debugging and inspecting WB-Labs applications in the browser.

- **WBC Inspector**: Real-time inspection of WBC component states, hooks, and data bindings.
- **Data-to-WBC Converter**: Context menu tools to transform highlighted snippets directly into WBC-compatible JSON.

## 💻 VS Code Extensions

Bringing the WB-Labs ecosystem into the world's most popular editor.

- **WBC Snippets & Intellisense**: Auto-completion for WBC properties and pipe syntax.
- **WBC Designer**: A side-panel preview that renders the active file using the WBC engine.
